### Project Descriptions

- show the credential list at the home page.
- able to record username/email and password for the domain website.
- able to edit existing credential and also delete the credential
- can enable biometric authentication and also set auto lock.
- can store credential in backend server to sync with other devices or in local only.

### Project Technical Details

- create using .Net 8 Web API with MAUI
